🧛 吸血鬼 (Vampire) — AI 虛擬人 Demo

網頁右下角會出現一位語音 AI 助理 —— 你可以「跟她說話」，她會聽懂、回答、開口說話並對嘴。
大腦可經由設置 DeepSeek API Key 啟用，對話內容與表情同步完全在瀏覽器端執行，API 金鑰保存在本機不外漏。

怎麼玩（請用 Chrome 開啟）

  1. 看右下角那位吸血鬼 —— 點她一下，她會打招呼、開口說話。
  2. 點 🎤 說話 → 准許麥克風 → 直接問她；不方便開麥也可以直接打字，例如「怎麼安裝」「怎麼換成你的角色」。
  3. 想讓她更聰明、能自然閒聊：點 🧠 輸入您的 DeepSeek API Key 啟用「AI 大腦」（金鑰保存在您的瀏覽器中，不外漏）。
